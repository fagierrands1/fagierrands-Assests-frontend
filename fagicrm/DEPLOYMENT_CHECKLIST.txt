TROUBLESHOOTING REFERENCE:
-------------------------
If 500 Error:
  - Check django_errors.log
  - Check ~/logs/error_log
  - Verify database credentials
  - Check ALLOWED_HOSTS setting
  - Restart application

If Static Files Not Loading:
  - Run: python manage.py collectstatic --noinput
  - Check staticfiles folder permissions
  - Verify .htaccess configuration
  - Restart application

If Database Error:
  - Verify database name includes username prefix
  - Check user has ALL PRIVILEGES
  - Test connection: python manage.py dbshell
  - Check password has no extra spaces

If Module Not Found:
  - Activate virtual environment
  - Run: pip install -r requirements.txt
  - Restart application
